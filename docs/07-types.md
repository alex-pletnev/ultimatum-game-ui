# 07. Types / DTO

Плейсхолдер до T-001 — типы придут вместе с чтением `frontend-integration/`.

## Куда положить

`src/api/types.ts` (или несколько файлов в `src/api/types/`, если контракт большой).

Правило: **типы фронта = зеркало backend'а**, не подгонять под удобство UI. Все UI-специфичные типы — отдельно, в `src/game/types.ts`.
