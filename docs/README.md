# ultimatum-game-ui — Документация

Набор справочных файлов для AI-агентов и разработчиков. Каждый файл — плотный, ориентированный на быстрый поиск: таблицы, ссылки `file:line`, минимум прозы.

## Индекс

| Файл | Что внутри |
|------|-----------|
| [01-overview.md](01-overview.md) | Обзор проекта, стек, точки входа |
| [04-components.md](04-components.md) | Карта UI-компонентов (плейсхолдер до T-002) |
| [05-api.md](05-api.md) | Контракт backend'а (плейсхолдер до T-001) |
| [07-types.md](07-types.md) | Типы данных / DTO (плейсхолдер до T-001) |
| [10-configuration.md](10-configuration.md) | Конфиги, env-переменные, dev-запуск |
| [11-known-gaps.md](11-known-gaps.md) | Что не реализовано, техдолги, риски |

## Как читать

- **Быстрый старт** → `01-overview.md`.
- **Нужен формат запросов к backend'у** → `05-api.md`.
- **Пишешь новый UI-компонент** → `04-components.md` (проверить, нет ли уже).
- **Заводишь env-переменную / меняешь конфиг** → `10-configuration.md`.
- **Ищешь, что «болит»** → `11-known-gaps.md`.
